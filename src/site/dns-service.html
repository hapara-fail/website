Instructions

Method 1: Standard Settings (For personal or unmanaged Chromebooks)

  1. Click the time in the bottom-right, then click the Settings gear ⚙️.
  2. Go to Network → Wi-Fi, and click the > arrow next to your current network.
  3. Expand the Network section.
  4. Under "Name servers," select Custom name servers.
  5. Enter 34.23.12.208 in all available name server fields.

Method 2: Network Internals (For managed or locked-down Chromebooks)

  1. Open a new tab and go to chrome://network.
  2. Click your current Wi-Fi network's name from the list. If the list is empty, click "Render Network Select". If your network isn't visible, click the empty area under the list.
  3. In the pop-up window, scroll down to the Name Servers section.
  4. Change the setting from "Automatic name servers" to Custom name servers.
  5. Enter 34.23.12.208 in all available text boxes.
  1. Open Control Panel, then go to Network and Internet → Network and Sharing Center.
  2. Click Change adapter settings on the left.
  3. Right-click your active connection (Wi-Fi or Ethernet) and select Properties.
  4. Select Internet Protocol Version 4 (TCP/IPv4) and click Properties.
  5. Select "Use the following DNS server addresses".
  6. Enter 34.23.12.208 in the Preferred DNS server box. If an "Alternate DNS server" box is available, enter the same address there as well.
  1. Open System Settings and go to the Network section.
  2. Select your active connection (e.g., Wi-Fi) and click the Details... button.
  3. Go to the DNS tab on the left.
  4. Remove any existing addresses in the list with the - button.
  5. Click the + button under DNS Servers and enter 34.23.12.208. Click OK.
  1. Open Settings and tap on Wi-Fi.
  2. Tap the blue "i" icon next to your connected network.
  3. Scroll down and tap Configure DNS.
  4. Select Manual and remove any existing DNS servers.
  5. Tap Add Server, enter 34.23.12.208, and tap Save.

Note: Steps may vary. Our DNS does not yet support the "Private DNS" feature.

  1. Open Settings and go to Network & internet → Wi-Fi.
  2. Tap the gear icon ⚙️ next to your network, then tap the pencil icon ✏️ to edit.
  3. Expand Advanced options and change "IP settings" to Static.
  4. Scroll down to the DNS 1 field and enter 34.23.12.208.
  5. In the DNS 2 field, enter the same address: 34.23.12.208. Tap Save.

Steps vary by distribution. These are general instructions for modern desktops using NetworkManager (Ubuntu, Fedora, etc.).

  1. Open Settings and navigate to the Network or Connections section.
  2. Find your active connection and click the gear icon ⚙️ to edit its settings.
  3. Go to the IPv4 tab.
  4. Disable the "Automatic" toggle next to DNS.
  5. In the DNS entry field, enter 34.23.12.208. Remove any other entries. Click Apply.
